Clean List Modal

Matches the existing modal pattern used by FriendsModal, KeyboardSettingsModal, etc. Same structure: p-0 override on the Modal content, fixed header with border-bottom, scrollable content area, and CloseButton in top-right. Border, shadow, radius, and backdrop color all come from the theme's CSS variables (--shadow-modal, --color-modal-bg, --color-modal-overlay, --color-border, --radius-lg, --border-width).

Scrollbar: Uses scrollbar-width: thin + scrollbar-color for Firefox, and ::-webkit-scrollbar rules for Chrome/Safari/Edge. This gives a thin, themed scrollbar on Windows/Linux instead of the chunky OS default, while macOS users see their normal overlay scrollbar.

UI/UX Decisions & Component Notes

How this modal is reached: User hovers the like button on SubmissionDetailPage → likers tooltip appears (see prototype/prototype-likers-popup.html) → user clicks "View all N likes" → this modal opens.

Modal styling: Uses the existing Modal component (src/components/shared/Modal.tsx). All border, shadow, radius, and backdrop values come from theme CSS variables (--color-modal-bg, --color-border, --radius-lg, --shadow-modal, --color-modal-overlay). No custom values. Follow FriendsModal structure: p-0, fixed header with border-bottom, scrollable flex-1 overflow-y-auto content area.

Reusable components — do NOT create new ones:
  • Modal (src/components/shared/Modal.tsx) — shared wrapper
  • CloseButton (src/components/shared/CloseButton.tsx) — lucide X icon, no background, hover color transition
  • FollowButton (src/components/social/FollowButton.tsx) — uses variant="ghost" with compact sizing. Handles follow/unfollow/loading/disabled states internally
  • AvatarImage (src/components/shared/AvatarImage.tsx) — Google avatar with initial-letter fallback

Follow button rules:
  • Only show for users the current user is NOT already following
  • Don't show for the current user's own row
  • After following, the button should disappear from that row
  • Button says "Follow" (not following) or "Unfollow" (following) — no "Following" intermediate state

Each liker row: Avatar + nickname (clickable, navigates to their profile) + optional FollowButton (right-aligned). Rows have subtle hover background. Nicknames truncate with ellipsis.

Cross-platform scrollbar styling (critical for Windows/Linux):
  • Firefox: scrollbar-width: thin; scrollbar-color: var(--color-border-light) transparent;
  • Chrome/Safari/Edge: ::-webkit-scrollbar 6px width, var(--color-border-light) thumb, rounded corners
  • Consider adding a scrollbar-thin utility in src/index.css alongside existing scrollbar-hide
